# Kernel

- Transparently compress and decompress when using `-save`/`-load`
- Numerous changes to the API, notably a streamlining of the `Cil` module;
  developers are aided in the migration through the use of ocamlmig.

# E-ACSL

- Support for a subset of inductive predicate definitions has been added.

# Eva

- The experimental Mthread plug-in, which uses Eva to analyze concurrent C programs, is now embedded within Eva and available in the open-source release.
- The Mthread analysis now supports interrupt handlers; see new option `-mt-interrupt-handlers`.
- Improvements to option `-eva-partition-history`

# Wp

- Assigns of aggregate objects (struct, union, array) are now based on memcpy (see `-wp-havoc`).
- Early prototype of Bytes memory model (for low-level bit manipulations, see `-wp-model bytes`, experimental)
- Early prototype of Region memory model (based on Region Analysis plug-in, see `-wp-model region`, experimental)

# Ivette

- Documentation of Ivette available in menu `Help -> Documentation`
- Visualization of global ACSL annotations
- New sidebar listing functions and global variables by source files
- The user can run an Impact analysis when right-clicking on a statement in the AST view.
